# Garment Defect Detection API

This project is a backend API for detecting defects on garments using computer vision and deep learning techniques.

## Goal
To build a production-ready AI-powered API that can detect fabric and garment defects from images.

## Tech Stack
- Python
- FastAPI
- OpenCV
- PyTorch (planned)
